Software Reverse Engineering to Bypass Registration

Many free to use programs usually have a premium use option. In this project, me and my team found a way using a dissasembler, IDA Pro, to locate the instruction that allows us to access the premium sections of the program. After using the dissasembler to carefully locate where the instruction address is, we then use a hex editor, HxD, to alter the instruction allowing us to use the premium sections of the program. Essentially most premium option programs have evolved making this method useless on some programs. However it is still possible. In this project we used a basic keylogger to reverse engineer.

  • Implementation: IDA Pro, x64dbg, HxD
  • Development Time: 2 Weeks

Camera Calibration, Stero Reconstruction, Augmented Reality

Throughout this course, we have learned a lot of key concepts in Computer Vision. In this project, we are tasked with taking pictures within a static scene and reconstructing the scene in a point cloud. Doing this requires many methods to get a 3D point cloud. First we must calibrate our camera. After calibration, we detect features in a pair of images. Within those pairs of images, we find matching features points. Next we use those matching feature points and our camera parameters to find the fundamental matrix. Finding the fundamental matrix then lets us find the camera pose of the images. From the camera pose we can get our camera matrices in which we can triangulate to get our 3D world points. And lastly from 3D points we can reconstruct our point cloud. Another part of this project is to detect a checkerboard in a scene and place a 3D wire frame cube. This requires some methods as well. For this problem we first obtain our Intrinsic and extrinsic parameters from our camera calibration. We use those parameters to create a camera matrix in which we can find 2D image points from our 3D wire frame cube points we made up. After finding the 2D image points, we overlay the images to create an augmented reality scene. In this project the language we used to code this all up is Matlab. This project used the reference of the math works matlab website to help accomplish this task.

  • Implementation: Matlab, OpenCV
  • Development Time: 3 Weeks
